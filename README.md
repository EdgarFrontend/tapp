# Telegram Mini App: Currency Converter (USD, RUB, AMD)

Современный интерактивный конвертер валют для Telegram Mini App с эффектом матового стекла (glassmorphism) и поддержкой тем Telegram. Позволяет мгновенно переводить суммы между долларами США (USD), российскими рублями (RUB) и армянскими драмами (AMD) при вводе.

## 🚀 Быстрый старт (Локальный запуск для браузера)

Для того чтобы просто посмотреть и протестировать интерфейс в обычном браузере, вам **не обязательно** настраивать Telegram-бота:

1. **Установите зависимости:**
   ```bash
   npm install
   ```

2. **Запустите локальный сервер:**
   ```bash
   npm run dev
   ```

3. **Откройте приложение:**
   Перейдите по ссылке [http://localhost:3000](http://localhost:3000) в любом браузере. Вы увидите полноценный интерфейс Mini App. Ввод в любое поле мгновенно пересчитает значения для других валют по реальным курсам!

## 🤖 Запуск внутри Telegram (Telegram Mini App Integration)

Чтобы запустить приложение как полноценный Telegram Mini App внутри мессенджера (на телефоне или компьютере):

### Шаг 1: Создание бота и получение токена
1. Откройте Telegram и перейдите к [@BotFather](https://t.me/BotFather).
2. Создайте нового бота с помощью команды `/newbot` и скопируйте его токен (API Token).

### Шаг 2: Настройка HTTPS туннеля (Ngrok)
Telegram требует, чтобы все Mini Apps работали строго по защищенному протоколу **HTTPS**. Для локальной разработки используйте `ngrok` или аналог:
1. Установите и запустите ngrok на порту `3000`:
   ```bash
   ngrok http 3000
   ```
2. Скопируйте сгенерированный HTTPS URL (например, `https://a1b2-34-56-78.ngrok-free.app`).

### Шаг 3: Настройка переменных окружения
1. Создайте файл `.env` в корневой папке проекта (или скопируйте `.env.example`):
   ```bash
   cp .env.example .env
   ```
2. Заполните `.env` вашими данными:
   ```env
   TELEGRAM_BOT_TOKEN=ваш_токен_от_botfather
   PORT=3000
   MINI_APP_URL=https://ваш_адрес_из_ngrok.ngrok-free.app
   ```

### Шаг 4: Привязка Mini App в BotFather
1. В чате с [@BotFather](https://t.me/BotFather) отправьте команду `/newapp`.
2. Выберите вашего бота.
3. Укажите название приложения (например, `Currency Converter`).
4. Укажите описание.
5. Загрузите картинку/гифку (если попросит, или пропустите).
6. Введите **Web App URL** — вставьте ваш HTTPS URL от `ngrok` (например, `https://a1b2-34-56-78.ngrok-free.app`).
7. Укажите короткое имя для ссылки приложения (например, `converter`).
8. В конце BotFather выдаст вам ссылку вида `t.me/your_bot/converter`.

*(Опционально)* Чтобы сделать кнопку запуска Mini App постоянной в чате с ботом:
1. В BotFather отправьте `/mybots` -> Выберите бота -> **Bot Settings** -> **Menu Button** -> **Configure menu button**.
2. Вставьте ваш HTTPS URL из ngrok и введите текст для кнопки (например, `Открыть Конвертер`).

### Шаг 5: Запуск проекта
1. Запустите бэкенд и бота:
   ```bash
   npm start
   ```
2. Перейдите в Telegram к вашему созданному боту, нажмите `/start` или кнопку меню. Откроется красивый интерфейс конвертера!

## 🛠️ Особенности реализации
- **Node.js + Express**: Быстрый и легкий сервер для раздачи веб-интерфейса.
- **Telegraf.js**: Профессиональный фреймворк для интеграции с Telegram Bot API.
- **Кэширование курсов**: Запросы к внешнему API кэшируются на 10 минут, чтобы предотвратить блокировки и обеспечить высокую скорость отклика.
- **Telegram WebApp SDK**: Автоматическое считывание цветовой схемы (светлая/темная тема) Telegram и использование встроенной тактильной отдачи (Haptic Feedback) при кликах.
